# ETF收益率预测与可视化系统（教学演示版）

## 项目简介

本项目展示了完整的金融数据分析流程，包括数据采集、特征工程、模型构建、模型评估和可视化展示。适合作为金融大数据课程设计的教学演示案例。

⚠️ **重要提示：本系统预测结果仅供参考，不能用于实际投资决策！**

## 功能特点

### 教学意义
- ✅ **方法论完整**：数据采集 → 特征工程 → 模型构建 → 模型评估 → 可视化展示
- ✅ **模型对比分析**：线性回归与随机森林双模型对比
- ✅ **工程实践**：Python爬虫 + React前端完整技术栈
- ✅ **可视化能力**：7个交互式可视化页面

### 技术特点
- 📊 数据采集：使用AKShare采集44只ETF的54,132条日频数据
- 🔧 特征工程：计算MA、MACD、KDJ、RSI等30个技术指标
- 🤖 模型构建：线性回归 + 随机森林双模型对比
- 📈 模型评估：MAE、RMSE、R²多维度评估指标
- 🎨 可视化界面：React + TypeScript + ECharts交互式界面
- 🛡️ 反爬机制：随机延迟、UA轮换、自动重试

## 实际局限性

### 模型局限
- ❌ 模型简单：线性回归和随机森林无法捕捉市场的复杂非线性关系
- ❌ 特征不足：仅30个技术指标，缺少新闻情感、宏观经济数据等关键因子
- ❌ 数据量有限：54,132条数据对于金融预测来说样本量偏小
- ❌ 缺少时序模型：没有使用LSTM、Transformer等时序深度学习模型

### 市场复杂性
- ⚠️ 政策变化：央行政策、监管政策、国际政策等，模型无法预测
- ⚠️ 突发事件：疫情、战争、自然灾害等黑天鹅事件
- ⚠️ 市场情绪：投资者恐慌、贪婪等心理因素
- ⚠️ 资金流向：大资金进出、机构操作等
